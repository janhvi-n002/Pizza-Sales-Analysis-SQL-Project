## Project Overview
This project analyzes pizza sales data using SQL to understand sales trends,
popular pizzas, customer behavior, and revenue performance.
## Dataset
The dataset includes the following tables:
- orders
- order_details
- pizzas
- pizza_types
## Queries Performed
- Total orders and total revenue analysis
- Top-selling pizzas by quantity and revenue
- Sales analysis by pizza category and size
- Peak ordering hours analysis
- Daily and monthly sales trend analysis
## Tools Used
- SQL (MySQL)
## Project Structure
Pizza-Sales-SQL-Project/
│── datasets/

## Key Insights
- Large-size pizzas generate the highest revenue
- Evening hours have the maximum orders
- Classic category pizzas are the most popular
